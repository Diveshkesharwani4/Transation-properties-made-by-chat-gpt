Git

The Version Control System

  • Installed locally on your machine
  • Tracks history of changes
  • Works offline
  • Handles branching & merging

GitHub

The Cloud Platform

  • Hosted on the web
  • Backup for your local code
  • Collaboration (Pull Requests)
  • Social coding network

The Basic Workflow

From your laptop to the cloud.

git add → git commit → git push

Command Cheat Sheet

Command Description
git init
Starts a new Git repository in the current folder.
git clone [url]
Downloads a repository from GitHub to your computer.
git status
Shows modified files and what is staged for commit.
git add .
Stages all changed files.
git commit -m "msg"
Saves the staged snapshot with a message.
git push origin main
Uploads local commits to the GitHub repository.
git pull
Updates your local code with changes from GitHub.
